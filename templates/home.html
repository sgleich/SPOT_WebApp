SPOT Microbial Eukaryotes

Welcome to the San Pedro Time-Series (SPOT) interactive web app! This web app loads, subsets, analyzes, and visualizes 18S amplicon sequence variant (ASV) data that were collected during monthly sampling procedures at the USC SPOT site from 2003-2018.

The ASVs used in this web app were created using DADA2 (Callahan et al. 2016) and taxonomy was assigned to the ASVs using the PR2 database (Guillou et al. 2012). The ASV dataset that was used in this web app was prefiltered so that 1.) singletons and doubletons were removed and 2.) ASVs were removed when a given ASV had 0s for > 90% of the samples at a given depth.
